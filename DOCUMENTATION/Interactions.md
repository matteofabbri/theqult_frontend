# Interazioni e Flussi Operativi

## 1. Sistema di Voto
- Universalmente applicabile a Post e Commenti.
- Gli utenti anonimi votano tramite un `anon-id` persistente nel browser per prevenire voti doppi senza login.

## 2. Economia dei Kopeki
- **Buy/Sell**: Conversione EUR <-> Kopeki (10k K = 1€).
- **Paywall**: Sblocco di contenuti esclusivi (ProfilePost) o accesso a Board a pagamento.
- **P2P**: Trasferimento diretto di denaro tra utenti via chat.

## 3. Messaggistica
Supporta tre tipi di contenuto:
- Testo semplice (opzionalmente criptato).
- Allegati multimediali.
- Trasferimenti di Kopeki.
- Inviti formali a Board private.

## 4. Pubblicità
Le board agiscono come marketplace:
- Gli utenti creano campagne (CPC/CPM).
- Gli admin della board approvano e guadagnano dai click/view.
